# 第五章：少一丁

## 第一节

第五日清晨，许衡刚把《仓耗杂记》塞进旧布包，葛小满就来了。

他这次没有蹲在灶边，而是站在院门口，表情很严肃。

“今日饴还出院子吗？”

许衡看了看天色，又看了看灶。

早饭还没做完，余温倒是有，可他今日要去旧渠。旧渠不比东坡近，且原身就是在修渠时淋雨高烧，最后把他这个外来魂魄换了进来。

从某种意义上说，旧渠算他的穿越事故现场。

许衡道：“今日不试。”

葛小满的脸立刻垮了。

“为何？”

“我要去旧渠。”

葛小满想了想：“饴怕渠？”

“饴不怕。”许衡说，“我怕。”

葛婶从墙后探头：“你倒知道怕。旧渠那地方，泥能吃鞋，风能灌骨头。你前些日子不就是在那里淋出病来的？”

许录正在灶边盛粥，闻言动作顿了一下。

许衡看见了。

屋里屋外忽然安静了半息。

原身的病不是一行背景。对许录来说，那是儿子昏了三日；对葛婶来说，是邻家小郎差点没醒；对许衡来说，是他睁眼来到这个世界的入口。

他低声道：“我会小心。”

许录把粟饼递给他，又多塞了一小块干姜。

“若下雨，便回。莫硬撑。”

“陈计吏那边……”

“你先活着。”许录说。

这句话不重，却让许衡没法接。

葛婶在旁边点头：“对。县里缺人是县里的事，你若死了，许家就缺人了。”

许衡本想说“我只是去看，不是去扛土”，但话到嘴边又咽了回去。

因为他忽然想到韩石当年大概也不是一开始就觉得自己会死。

葛小满看气氛不对，悄悄把草茎递给许衡。

“给你。路上打泥怪。”

许衡接过那根毫无战斗力的草茎，认真收进布包边缘。

“多谢。”

葛婶看得直摇头：“一个敢给，一个敢收。”

许衡笑了笑。

这点笑意很轻，却让院里的沉气散了些。

出门时，他回头看了一眼小陶盂。

今日甜水停了。

问题簿上会少一条试验记录。

可旧渠那边，可能有更多东西等着他记。

## 第二节

县衙侧屋里，陈简已经把几片木牍摆好。

一片是东坡核牍。

一片是铁刃进度。

一片是旧渠今日役册。

许衡进门时，伍仓正低头偷看那片役册，嘴里念念有词。

“东坡里二十七，南沟里十九，北槐里二十二……这哪是修渠，这是把半个县往泥里赶。”

陈简抬眼。

伍仓立刻闭嘴。

陈简把旧渠役册推到许衡面前。

“今日随修渠吏罗直去旧渠。你不管役，不判责，只看三件事。”

许衡坐下。

陈简伸出一根手指。

“一，册上多少。”

第二根。

“二，实到多少。”

第三根。

“三，少的那几人，渠上怎么补。”

许衡点头。

陈简又道：“还有铁刃。”

“四口可用，一口裂纹待验，一口未磨。”

“今日看它们到了渠上算几口。”

许衡心里一动。

县衙里写“可用四”，铁匠铺里摆着四口，到了渠边，可能又会变成另一种数。

伍仓在旁边叹气：“你们这数分得越清，我跑得越多。”

陈简没有理他。

一个身材敦实的中年人从门外进来，脸上有一道旧疤，从眉尾斜到颧骨。他穿着短衣，袖口卷起，手上全是老茧，腰间挂着竹牌。

伍仓立刻精神起来。

“罗渠吏。”

罗直扫了许衡一眼。

“就是他？”

陈简道：“许衡。随你去看旧渠。”

罗直皱眉：“病刚好那个？”

许衡：“是。”

罗直看他细胳膊细腿，脸上写满“不堪用”。

“能下泥？”

许衡想了想：“能看泥。”

伍仓差点笑出声。

罗直没笑。

“旧渠上不是县衙侧屋。有人吵，有人懒，有人真病，有人装死。你若站不住，别挡路。”

许衡点头：“我尽量不挡。”

罗直转身就走。

伍仓小声道：“罗直人不坏，就是说话像铁锨。”

许衡道：“挺好，至少用途明确。”

伍仓看了他一眼，忽然觉得这两人可能会相处得不错。

## 第三节

旧渠在桑里县东南。

比东坡路更开阔，也更乱。

远远看去，渠沟像一道被人撕开的土口，横在田边。春水从上游一点点渗下来，被临时堵在几处土坝后。渠边堆着泥、石、木桩、竹筐，还有一群脸色不怎么好的役夫。

他们看见罗直来了，吵声立刻低了些。

只低了一些。

“铁刃还少！”

“北槐里的人没到齐，凭什么我们先下？”

“昨日说今日换班，我家田还没翻！”

“那边水口塌了，不先堵，下午就白干！”

罗直站上土坡，吼了一声。

“闭嘴！”

声音很大。

渠边安静了一瞬。

许衡站在罗直身后，觉得这大概就是旧渠现场的主要治理工具。

罗直把役册往伍仓怀里一塞。

“点名。”

伍仓苦着脸接过，开始喊。

“东坡里，韩伯常！”

“在！”

韩伯常今日也来了，站在队尾，脸色比昨日更黑。

“韩槐！”

一个三十来岁的汉子应声。他肩宽背厚，手里拎着木锨，看向许衡时眼神有些防备。

许衡想起韩伯常牍上那行：韩槐代半日。

伍仓继续喊。

“韩石！”

渠边没人应。

这个名字落下来时，风声忽然显得很清楚。

许衡看向役夫队列。

有人低头，有人看韩伯常，有人看罗直，也有人不耐烦地啧了一声。

伍仓抬头，看向罗直。

罗直看向许衡。

许衡取出昨日核牍。

“韩石，亡实。役空。”

罗直脸上没有惊讶。

显然他早就知道。

“役空归哪？”

许衡答不上来。

罗直也没等他答，转头对韩伯常道：“今日东坡少这一人，土筐谁补？”

韩伯常咬牙：“东坡已到能下渠二十三。病的病，逃的逃，死的死。再补，就只能让女人和孩子来。”

队伍里有人立刻道：“谁家没女人孩子？我家田里也缺人！”

韩槐沉声道：“去年我已替半日。今年还替，谁替我家田？”

另一个役夫冷笑：“你不替，难道让韩石从坟里出来替？”

韩槐脸涨红，往前一步。

罗直一把抓住他的肩。

“都给我闭嘴。吵能堵水？”

许衡站在旁边，手心出汗。

册上一个名字空出来，落到渠边，就不是“删”或“不删”的问题。

是一个土筐的位置空着。

是这一段渠谁多挖一锨，谁晚回家半刻，谁家田再耽一天。

灰字浮出：

```text
【旧渠役册：东坡里旧额二十七】
【今日可下渠：二十三】
【状态：缺口未归属】
```

许衡没有说话。

罗直看他一眼：“记。”

许衡低头写：

```text
东坡旧额二十七。
今日可下渠二十三。
韩石亡实，役空未归。
渠上缺土筐一位。
```

写到“土筐一位”时，他停了一下。

这个说法不雅。

但比“少一丁”更像他刚才看到的东西。

## 第四节

铁刃也到了。

四口。

四口磨好的铁刃被修渠吏的副手抱来，放在土坡上。刃口亮，木柄旧，远看像四件终于赶到现场的救兵。

役夫们围过来，脸色却没好多少。

罗直问：“还有两口呢？”

副手道：“一口裂纹待验，一口未磨。铁匠说炭湿，火起不稳。”

罗直看向许衡。

许衡把昨日牍拿出来。

“与铁匠铺所记相合。”

罗直骂了一句。

“相合有屁用。渠上还是四口。”

许衡被骂得很平静。

因为他说得对。

县衙里“相合”，只是说明这次没有凭空少数。旧渠上“四口”，才是今日能用的工具。

四口铁刃很快分下去。

南沟里一口，北槐里一口，东坡里一口，水口急段两口。

许衡听到这里，抬头。

“这不是五口？”

罗直看他。

副手也看他。

伍仓立刻往旁边挪了一步，像怕被算术波及。

罗直道：“水口急段要两口，从北槐里先借一口。”

北槐里一个汉子立刻骂：“凭什么借我们？”

罗直指着水口：“那边塌了，先堵那边。你若不服，站水里同它讲理。”

汉子闭嘴。

许衡在木牍上写：

```text
铁刃到渠：四。
分配：东坡一、南沟一、水口二。
北槐暂缺。
```

他想了想，又在旁边写：

```text
县中可用数与渠上分配数不同。
```

伍仓凑过来，小声道：“你这句陈计吏看了会高兴。”

“为何？”

“因为又多一件麻烦能归册。”

旧渠开始动工。

许衡站在土坡边，看着役夫们下泥。

有铁刃的地方，泥土被切开得快些。没铁刃的地方，木锨打在湿硬的土上，发出闷响，半天才撬开一块。水口那边最急，两个壮劳力轮着挖，旁边的人用筐运泥，脚下全是滑浆。

东坡里只有一口铁刃。

韩槐拿着木锨，被罗直安排去补韩石那个空出来的土筐位。他没有再吵，只闷头干活。

半刻后，他额头上全是汗。

许衡看着他把一筐泥递给旁边的人，又转身去接下一筐。

韩石死了。

韩石那一筐泥没有死。

它只是换到了韩槐手里。

许衡低头想写，又停住。

这句话太像感慨。

陈简大概会让他重写。

他最后只写：

```text
韩槐补土筐位，半刻内加运三筐。
```

少一分把握，便少写一分定话。

## 第五节

旧渠上午出了第一次乱子。

不是大乱子。

水口那边一段湿土忽然塌下去，泥浆滑进渠底，差点把刚立起的木桩冲歪。罗直吼着让人补，两个拿铁刃的壮劳力立刻下去，旁边运泥的人也跟着加快。

东坡这边被抽走两人去水口。

原本就少一位的土筐队，立刻断了一截。

韩槐一人补不过来，另一个年轻汉子接上。

他身材高，肩膀宽，动作很快，搬筐时不怎么说话。伍仓在旁边告诉许衡：

“田烈，南沟里的人。力气大，脾气也大。前日还和县尉吵过，说他家田再误，要把县尉家门口种粟。”

许衡看过去。

田烈一连扛了四筐，脸色沉得像渠底的泥。

有人喊：“田烈，慢点！”

田烈没理。

第五筐时，他脚下一滑，整个人往旁边歪了一下。泥筐砸在地上，泥水溅起，旁边一个瘦役夫被带倒，膝盖磕在石头上，疼得倒吸气。

罗直大步过去，一把拽起田烈。

“急着投胎？”

田烈甩开他的手。

“水口要土，你喊得比谁都急。”

罗直指着他鼻子：“我要土，不要你摔死在渠里。”

田烈冷笑：“摔死了，册上也能多留两年？”

这话一出，周围安静了一下。

许衡握笔的手停住。

罗直的脸沉下来。

伍仓立刻低声道：“田烈，少说两句。”

田烈没再说话，只弯腰去扶那个膝盖磕伤的瘦役夫。

瘦役夫摆摆手：“无事，无事。”

可他站起来时，右腿明显不敢用力。

许衡看着这一幕。

少一丁。

少两口铁刃。

水口一塌。

原本每个人只需要多一点的事，很快就变成有人多扛几筐，有人摔一跤，有人膝盖见血。

灰字浮出：

```text
【临时调配：增加】
【单人负荷：上升】
【轻伤：一】
```

许衡没有写“因少一丁导致”。

他还不能这么写。

他只写：

```text
水口塌，东坡抽二人。
韩槐、田烈补土筐。
田烈滑倒，带倒一人，膝伤。
```

写完后，他忽然觉得这片木牍很重。

因为它没有骂谁。

但谁看了都不会舒服。

罗直处理得很快。

他让膝伤的瘦役夫坐到坡边，叫另一个人替上，又从自己腰间解下一小包药灰，撒在那人膝上。

许衡有些意外。

罗直看见他的眼神，没好气道：“看什么？真伤了，下午更少人。”

这话说得难听。

但药灰撒得很稳。

许衡把这也记下：

```text
罗直以药灰止血，令伤者暂歇。
```

伍仓小声道：“你连这个也记？”

许衡道：“记。”

“为何？”

许衡看着坡边那个抱着膝盖的役夫。

“不记，下午他若又被点下渠，就没人知道他上午伤过。”

伍仓怔了一下，嘴动了动，最后没说话。

## 第六节

午后，旧渠总算稳住了。

稳住，不是修好。

只是水口暂时不塌，泥筐能继续走，四口铁刃还没断，罗直的嗓子也还没哑。

许衡坐在土坡边，手腕酸得发麻。

他今日没下泥，只写。

可他觉得自己比抄册那日更累。

因为县衙里的错，写在竹简上；旧渠里的错，写在人身上。竹简错了可以重抄，人摔一跤，膝盖不会因为你改字就不疼。

罗直走过来，看了眼他膝上的木牍。

“写了多少？”

许衡把木牍递过去。

罗直不识太多字，看得慢。伍仓在旁边替他念：

```text
东坡旧额二十七。
今日可下渠二十三。
韩石亡实，役空未归。
渠上缺土筐一位。
铁刃到渠四，分配后北槐暂缺。
水口塌，东坡抽二人。
韩槐、田烈补土筐。
田烈滑倒，带倒一人，膝伤。
罗直以药灰止血，令伤者暂歇。
```

念到最后一句，伍仓看了罗直一眼，似乎想笑又不敢。

罗直脸上没什么表情。

“最后一句删了。”

许衡抬头。

“为何？”

“我不求你替我记好名。”

许衡道：“这不是好名。”

罗直皱眉。

“那是什么？”

“这是伤者为何暂歇。”

罗直盯着他看了一会儿。

许衡补了一句：“若下午有人问他为何少干半日，这句有用。”

罗直不说话了。

过了一会儿，他把木牍丢还给许衡。

“随你。”

这两个字听起来不耐烦。

但没有再让他删。

傍晚回县衙，陈简看完木牍，停在“渠上缺土筐一位”和“膝伤”两处。

“你今日看见了？”

许衡点头。

“看见什么？”

许衡想说：看见韩石死了，韩槐多运三筐，田烈滑倒，瘦役夫膝伤，罗直骂人也撒药，铁刃四口到了渠上还是不够。

可这些都是事。

陈简问的似乎不是这些。

他沉默许久，最后道：“少的不是名字。”

陈简看着他。

许衡继续说：“少一个名字，渠上少一双手。若不写清，旁人就会用自己的手补。”

陈简没有评价。

他只在木牍旁加了一行：

```text
旧渠实到牍，留案。
```

许衡松了口气。

伍仓在旁边小声嘀咕：“留案留案，案上越来越挤，渠上还是越来越缺。”

陈简抬眼。

伍仓立刻闭嘴。

回到家时，天已经黑了。

葛小满照旧在院门口等。

看见许衡一身泥点，他先问：“今日饴呢？”

许衡把布包放下，摊开两手。

“今日没有。”

葛小满失望得很明显。

葛婶在旁边道：“人都快成泥人了，你还惦记饴。”

葛小满小声道：“他说回来补。”

许衡看着他。

他说过吗？

他说的是，若有余温就试，若没有明日补。

小孩子记得很准。

“今日没有余温。”许衡说，“明日补。”

葛小满盯着他：“真的？”

许衡点头。

“真的。”

这一次，他说得比昨日慢。

夜里，许衡没有做甜水，只在问题簿上写了两行：

```text
今日无试。
原因：旧渠归晚，无余温。
明日补。
```

写完后，他又把旧渠实到牍的副记放在旁边。

```text
韩石亡实，役空未归。
渠上缺土筐一位。
田烈滑倒，膝伤一人。
```

两片木牍摆在一起。

一片记着没兑现的小甜水。

一片记着没人归属的一位役空。

许衡看着看着，忽然想起师艺那句话。

别只看没超限，先看它为什么不收敛。

他闭了闭眼。

这个世界没有报警。

只有一个孩子等着甜水，一个死人还在册上，一个活人替他多扛了几筐泥。

油灯跳了一下。

许衡提笔，在旧渠副记最后添了一句：

```text
明日问：伤者半日如何记。
```

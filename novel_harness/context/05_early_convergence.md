# 早中期情节收束：废庙与铁炉

## 目标

第一卷早中期所有“小技术”和“小买卖”情节，都应自然收束到“废庙与铁炉”这一主题转折点。该转折点不是主角突然开悟，而是他积累了足够多的小挫败后，第一次意识到：物件、产业、制度、共同预期之间存在关系。

注意：废庙与铁炉是第一次大转弯，不是最终理论完成点。它之后还要经过役棚、村社/里、县级工坊/铁炉、战时/灾年/迁民小邑等逐级管理失败，主角才有资格总结衡学。

## 收束链条

```text
穿越醒来
→ 家庭存粮、役册重核
→ 系统显示基础统计
→ 主角用现代小技术尝试改善处境
→ 制糖、造纸、肥皂、蒸酒带来小钱和笑点
→ 原料、柴价、锅具、工匠、关津、税吏、豪强逐渐卡住他
→ 主角发现技术利润总被制度节点截流
→ 县外铁官作坊扩张，铁器流入农具和兵器
→ 旧宗庙旁的新铁炉形成强烈对照
→ 公羊度出场，谈礼崩不是钟坏，而是旧图装不下新世道
→ 系统解锁【未归之账】
→ 主角明白小术赚来的钱只能购买下一阶段试验资格
→ 后续用钱、名声和账本能力接触役棚、村社、铁炉，最终走向衡山试验田
```

## 早期支线如何指向废庙与铁炉

### 白糖线

表面功能：第一个商业爽点。

收束作用：

- 证明主角有技术优势。
- 暴露原料供应、关津盘剥、豪强入股、税目混乱。
- 让主角知道“商品利润不等于可支配收益”。

### 造纸线

表面功能：工坊试错和轻喜剧。

收束作用：

- 暴露标准化、良率、匠人训练、批量生产问题。
- 让主角知道“发明物件不等于建立产业”。

### 肥皂线

表面功能：生活改善和误会笑点。

收束作用：

- 暴露古代消费能力、卫生观念、税吏分类、市场容量。
- 让主角知道“现代需求不能直接复制到古代”。

### 修渠/役册线

表面功能：基层压力。

收束作用：

- 把主角从商人小利拉回国家基层账册。
- 为后续“役损账”和“未归之账”埋伏笔。
- 这是未来第一步“役棚，小系统，快反馈”的入口，不要在第一卷直接写成完整治理理论。

### 铁器线

表面功能：时代背景。

收束作用：

- 让主角看见底层生产力变化。
- 连接旧礼崩坏与新制度未立。
- 成为废庙与铁炉场景的物质核心。
- 后续要承担第三步“县级工坊/铁炉，多目标冲突”，并为伍仓等高风险承担者的命运埋下基础。

### 赚到的钱

表面功能：系统文和穿越文的早期爽点。

收束作用：

- 让主角可以垫粮、买工具、雇匠、赎役，获得试验资格。
- 让主角误以为钱能买出秩序。
- 后续通过流民、役棚、铁炉证明：钱只能买第一顿饭，买不了长期信任。

## 语气控制

早中期可以欢脱，但要有方向。

推荐节奏：

```text
小发明成功
→ 主角得意
→ 古代现实用荒唐方式打脸
→ 主角嘴硬或吐槽
→ 系统冷淡补刀
→ 留下一个更深的问题
```

例：

```text
主角：这次我算过火候，绝不会糊。
系统：【焦化比例：上升】
鲁墨：许兄，你这糖为何有炭香？
```

笑点之后要回到问题：柴价、锅具、批量稳定性、运输损耗、税吏盘剥。

## 第一卷草蛇灰线

第一卷重点保留三条即可：

1. **不收敛的曲线**：从现代设备曲线，转向社会小误差长期累积。
2. **洁净服/问题提单**：从现代工程流程笑点，转向作坊问题簿和制度化记录。
3. **缺口铁刃**：从家中农具损耗，转向铁匠铺、铁官作坊、废庙与铁炉。

不要再额外密集添加大伏笔，避免读者负担过重。

## 禁止

- 不要一开始就让主角说“社会是控制系统”。
- 不要让废庙与铁炉突兀出现，必须由前面的小技术挫败自然推到那里。
- 不要把前期写成纯苦难。主角应有小胜、小钱、小得意，也有小倒霉。
- 不要让理论压倒角色互动。
- 不要使用生硬技术梗。笑点应来自角色职业惯性和古代现实错位。

## 长期框架约束

- 主线应遵循：现代小术赚钱 -> 小术失灵 -> 钱和名声换取试验资格 -> 役棚/村社/铁炉/流民小邑四级失败与修正 -> 衡山试验田 -> 衡学 -> 外部失稳下的扩张逐鹿。
- 现代物品复刻赚到的钱不能成为万能解法，只能让许衡垫粮、买工具、雇匠、赎役、招揽游民，并向诸侯换取一个没人愿意接的试验田。
- 四级认知阶梯：
  - 役棚：小系统、快反馈，许衡用现代平等和少受苦 target 试错，学到成本会转嫁。
  - 村社/里：季节系统、延迟反馈，学到今日仁政可能吃掉明年的种子。
  - 县级工坊/铁炉：中系统、多目标冲突，学到危险公共品需要风险分配、补偿、名誉和可信后路。
  - 战时/灾年/迁民小邑：大系统、真正 target，学到共同体要在冲击后仍愿意留下、生产、守约、生育、承担危险。
- 衡山命名应逐步演化：西横废里/横山废里 -> 衡山里 -> 衡山 -> 衡学。衡山不是天然仙山，而是废庙、铁炉、流民、田渠共同长出的制度根据地。
- 衡学对外呈现分两层：普通人可执行的低阶近似，如“有劳必记，有险必偿”；士人可辩论的高阶解释，如“衡者，非使万物齐也，使万物各得其所承”。
- 诸子百家应作为旧系统崩坏后的不同构建准则：儒为共同想象，墨为工程约束，法为执行器，道为系统边界，兵为外部扰动，农为底层输入，名为测量口径。
- 人物羁绊是主角理论内在动力。伍仓、陈简、许录、葛婶与孙子、公羊度等人要先有生活细节和具体关系，再内化为制度。
- 伍仓可在铁炉阶段后段因主角尚不成熟的系统、多重外部压力和高危任务叠加而死，推动伤养约、高危补偿、功簿继承等制度；不要写成廉价挡刀。
- 中后期系统可形成衡仪人格，带有现代白月光师艺的认知回声。她是否就是本人不完全确认；她负责陪伴、追问、校验、转译控制论，不提供万能答案，不转为实体女主或后宫线。
- 共同体稳态假说应在中后期由经历平滑推出：两条根公理为生息公理、代价归账公理；四个约束为测量不完备、信任有限、价值偏好需归账、外部扰动不可消失。
- “价值偏好需归账”替代“权力倾向外包成本”的窄说法：任何自认为善、义、公、仁、进、礼的行动，在施行前都要追问成本来源、利益去向和迟来反馈。
- 读者参与推导：核心理论命名前，必须先通过具体场景、物件、账目、失败和人物代价让读者隐约看见共同结构；主角总结应晚于读者直觉半拍。

- 东坡里正韩伯常出场。韩伯常承认韩石确亡，但因东坡里旧额二十七、实有约二十三，担心合死籍后空额无人补，所以一直暂缓。

- 第五章清晨，许衡因要去旧渠而停做甜水；葛小满追问，许衡承诺“今日没有余温，明日补”，进一步强化小承诺线。
- 许录提醒许衡旧渠危险，原身前些日子正是在修渠时淋雨高烧；许录说“你先活着”，葛小满给许衡一根草茎“打泥怪”，保留生活气。
- 陈简安排许衡随修渠吏罗直去旧渠，只看三件事：册上多少、实到多少、少的人渠上怎么补；另看铁刃到了渠上算几口。
- 修渠吏罗直出场：中年敦实，脸有旧疤，手有老茧，说话硬，负责旧渠现场。不是单纯恶吏，后续会给伤者撒药灰并让其暂歇。
- 旧渠现场：东坡里旧额二十七，今日可下渠二十三；点名韩石无人应，许衡据核牍写“韩石亡实，役空未归；渠上缺土筐一位”。
- 铁刃到渠只有四口。现场分配后：东坡一、南沟一、水口二，北槐暂缺。许衡意识到县中“可用数”与渠上“分配数”不同。
- 韩槐被安排补韩石空出来的土筐位，半刻内加运三筐。许衡只记具体事实，没有直接写感慨。
- 水口湿土塌落，罗直临时从东坡抽二人，土筐队断开。南沟里役夫田烈补土筐，连续扛筐后滑倒，带倒一名瘦役夫，瘦役夫膝伤。
- 罗直骂田烈“我要土，不要你摔死在渠里”，并给膝伤役夫撒药灰，令其暂歇。许衡坚持记下此事，因为它能解释伤者下午为何少干半日。
- 旧渠实到牍留案，内容包括东坡旧额二十七、可下渠二十三、韩石亡实役空未归、渠上缺土筐一位、铁刃到渠四、田烈滑倒带倒一人膝伤、罗直止血令暂歇。
- 陈简问许衡看见什么，许衡只说“少的不是名字。少一个名字，渠上少一双手。若不写清，旁人就会用自己的手补。”仍未上升到完整理论。
- 第五章结尾，许衡没能给葛小满做甜水，只记“今日无试，原因：旧渠归晚，无余温，明日补”。同时记下“明日问：伤者半日如何记”。
- 第五章末许衡想起师艺“别只看没超限，先看它为什么不收敛”，这是现代记忆回声，不是系统人格出现。

## 禁止遗忘事项

- 主角不能早期全知。
- 公羊度启发前，主角只能模糊感到古代制度有问题，不能系统阐述。
- “未归之账”必须在旧宗庙与铁官作坊情节附近触发。
- “未归之账”触发后也不能直接完成衡学。废庙与铁炉只是第一次主题转折，后续还要逐级失败与修正。
- 主角早期可以真诚使用现代平等、人权、少受苦等直觉设定 target，但必须让具体反馈揭示这些 target 需要生产力、信用、补偿和执行结构承载。
- 系统人格不能早期出现。前期系统只统计；中期可出现回声；伍仓之死或同等重大事件后才可形成衡仪式对话人格。
- 衡仪不能替读者总结公理。她只能追问 target、成本归属、反馈接收者和价值偏好是否已校验。
- 类《三体》的公理化推演只能发生在主角与衡仪私下对话中；对古人公开传播必须转译为衡山六账、衡学六义、同约、功簿等时代语言。
- 当前系统尚未显示市价统计、工坊统计、田亩统计，需在主角接触市场/田亩/工坊后逐步解锁。
- 第一卷早中期的小技术、小买卖、小倒霉、小笑点都要自然收束到“废庙与铁炉”。
- 不要连续苦大仇深；早期需要保留生活气、机锋和轻松误会。
- 幽默要自然短促，避免“未经过滤的中药系统写进人生主循环”这类生硬技术梗。
- 每卷重点草蛇灰线控制在 1-3 条；第一卷暂定三条：不收敛曲线、洁净服/问题提单、缺口铁刃。

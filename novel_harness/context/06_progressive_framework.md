# 逐级顿悟与衡山框架

## 总体主线

主角的成长不能从一次谈话或一次役棚管理直接跳到完整理论。整部小说的骨架应是：

```text
现代小术赚钱
→ 小术被原料、燃料、关津、税吏、豪强、组织缺失卡住
→ 主角用钱、名声和账本能力换取试验资格
→ 役棚、村社、铁炉、流民小邑逐级失败与修正
→ 衡山成为制度试验田
→ 衡学从制度实践中被迫总结出来
→ 外部系统失稳，稳定系统被迫扩张，最终逐鹿
```

钱不是答案，只是门票。现代物品复刻带来的收益，应让主角获得垫粮、买工具、赎役、雇匠、招揽流民和向诸侯换取试验田的资格；但钱很快会暴露上限：它能买第一顿饭，买不了明年的信任。

## 四级认知阶梯

### 第一步：役棚

性质：小系统，快反馈。

典型对象：修渠役棚、筑堤、运石、修仓。

主角早期 target：

- 不许役吏随便打人。
- 同工同食。
- 病者可以休息。
- 按天轮换，避免累死。
- 账目公开，人人有申诉机会。

必须保留的现代直觉：主角此时真诚相信人权、平等、少受苦、少压迫这些目标，而不是一开始就变成冷酷制度家。

失败或打脸：

- 壮劳力吃不饱，弱劳力吃不完。
- 有人假病，有人真病却不敢说。
- 工期被拖慢，县里催工，兜底压力回到主角身上。
- 家中农时被误，春耕缺人的户偷偷逃役。
- 最老实的人承担最多，因为他们不闹、不跑、不装病。

主角可得到的低阶认知：

```text
苦不会因为我心善而消失，只会换一个人背。
```

禁止：不要在这一步让主角总结出完整“社会控制系统”理论。

### 第二步：村社/里

性质：季节系统，延迟反馈。

典型对象：村社春耕、粮仓互济、水渠排班、役期调整。

主角 target：

- 不误农时。
- 保护弱户。
- 重排役期。
- 粮仓互济。
- 铁器、水、牛力优先给最需要的人。

失败或打脸：

- 救急吃掉种子粮。
- 强户觉得被惩罚，开始隐藏库存或消极合作。
- 弱户名额被冒占。
- 当季看似仁政，秋后才显现代价。
- 大家不知道新规能持续多久，所以都留后手。

主角可得到的低阶认知：

```text
今日看似公平的分配，可能在明年变成新的饥荒。
人不是不懂道理，是没人敢把明年的命押在今天的好话上。
```

### 第三步：县级工坊/铁炉

性质：中系统，多目标冲突。

典型对象：县级铁官外坊、废庙旁旧铁炉、炭料、匠人、役夫、军需、农具。

主角 target：

- 提高铁器产量。
- 保证农具供应。
- 减少炉伤。
- 不让匠人跑光。
- 不让役夫被一次性消耗。
- 满足军需但不抽干来年生产。

失败或打脸：

- 军方要刀矛，农户要犁铧。
- 县令要短期政绩，村社要长期活命。
- 匠人要安全和传承，役夫要活命和回家。
- 提高产量会增加事故，降低事故会降低产量。
- 给高危岗位补偿，低危岗位会不满；不给补偿，高危岗位没人愿意长期上。

主角可得到的低阶认知：

```text
危险公共品不是靠勇敢自然生产的。
它需要风险分配、补偿、名誉、惩罚和可信后路。
```

这一阶段适合发生关键人物死亡，尤其是伍仓，作为主角从“会管理”走向“必须立约”的触发点。

### 第四步：战时/灾年/迁民小邑

性质：大系统，真正 target。

典型对象：战后废里、灾荒流民聚集地、边境缓冲小邑、废庙铁炉旁荒村。

进入方式：

- 主角用前期小术赚到的钱垫粮、买工具、赎役、招匠、招揽游民。
- 向诸侯或县君换取一块没人愿意接的试验田。
- 条件应带毒：三年不入税、逃户过半、铁炉停废、盗贼往来；若活之则有名，若败则自任其罪。

主角 target：

```text
让一群失去旧秩序的人重新愿意留下、生产、守约、生育、承担危险。
```

失败或打脸：

- 流民不信官府，也不信承诺。
- 来源复杂，有农户、逃兵、匠人、寡妇、孤儿、罪户、破产小商。
- 钱能让人留下几天，不能让人把家安在这里。
- 秩序太松会散，秩序太紧会逃。
- 对内要安民，对外要防盗、应战、交涉、纳税。

主角可得到的高阶认知：

```text
治理不是让某个指标最大化。
治理是让一个共同体在冲击之后仍然愿意继续存在。
```

## 衡山命名演化

“衡山”可以使用。现实中有同名地名，但本书为架空战国，重复不构成问题。为了避免名字一出场就太正统、太仙侠，应让它逐步长成。

推荐演化：

```text
县册旧名：西横废里 / 横山废里
乡人口语：横山
主角立约后：衡山里
制度成形后：衡山
学派成形后：衡学、衡山之约、衡山功簿、衡山炉
```

命名触发可由公羊度或陈简提出：

```text
横者，拦也；衡者，称也。此地若成，不当名横，当名衡。
```

衡山的空间象征：

- 山上废庙：旧共同想象崩坏。
- 山腰铁炉：新生产力与危险公共品。
- 山下流民聚落：人口再组织。
- 山外田渠：粮食、农时、基本输入。

衡山不是修仙宗门，而是制度根据地。可以带一点山门气，但本质是收人入约、记功、养伤、分粮、开炉、垦田。

## 衡学的双层表达

衡学不能太早出现。它应在主角管过小城或准小城，并且制度救过灾、扛过战、吸收过流民后，被旁人追问“这些做法背后是什么道理”时被迫总结。

### 低阶近似

给普通人、役夫、农户、士卒、匠人的版本必须简单可执行：

```text
有劳必记。
有险必偿。
有利必约。
有令必验。
有伤必养。
有功必归。
```

### 高阶解释

给士人、官吏、诸子辩论时，可以带类似道家的复杂系统味道，但不能直接等同道家：

```text
万事不平则流，流久则偏，偏久则覆。
故治不在强平，而在知其所偏，使费有所归，险有所受，利有所约，名实相副。
```

核心句：

```text
衡者，非使万物齐也，使万物各得其所承。
```

衡学不是纯管理术，也不是纯道德说教。它必须给各阶层一个可理解、可期待、可执行的生活范本。

## 诸子百家的系统位置

外部世界必须逐步走向失稳。诸子百家不是背景板，而是在同一个崩坏时代里给出不同“新系统构建准则”。

- 儒家：礼、名分、共同想象。
- 墨家：节用、尚贤、工程、标准化，降低无效损耗。
- 法家：法、术、势、执行器、惩罚和强制动员。
- 道家：顺势、少扰、警惕过度控制导致反噬。
- 兵家：外部扰动、冲突环境、士气、欺骗、边界扩张。
- 农家：粮食、土地、农时、人口这些底层输入。
- 名家：名实错位，语言、身份和账目口径如何误导判断。

主角不应嘲笑诸子百家，而应逐步吸纳他们的局部正确性。

衡学后期可被写成：

```text
礼是共同想象，法是执行器，墨是工程约束，道是系统边界，兵是外部扰动，农是底层输入，名是测量口径。
```

但这类总结必须等主角经历足够多之后再出现。

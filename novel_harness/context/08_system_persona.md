# 系统人格：衡仪与白月光回声

## 总原则

系统人格可以在中后期出现，作为主角精神世界里的类女主存在。它的功能是陪伴、校验、追问、翻译现代控制论语言，而不是发放答案或替代现实人物羁绊。

前期系统必须保持冷静、短促、非人格化。只有当主角经历足够多现实失败、积累大量问题簿和未归之账，并在一次重大伤亡后无法独自承受复盘压力时，系统人格才逐步成形。

## 现代白月光：师艺

师艺是主角现代世界里的重要女性，可以是大学师姐、项目组控制算法同事，或半导体设备联调中的长期搭档。

她不必是正式恋人，但必须有白月光重量：

- 能听懂控制论、故障树、反馈延迟、目标函数、约束条件。
- 与主角有未完成关系：互相欣赏、经常互怼、差一点表白，或主角一直没敢说出口。
- 她不只是漂亮回忆，而是主角现代思维的一部分。
- 她曾经提醒过主角：指标收敛不等于系统正确，人不是干扰项，目标函数是谁定的也要被追问。

可用现代草蛇灰线：

```text
别只看没超限，先看它为什么不收敛。
你总想着让曲线收敛，可人不是干扰项。
目标函数是谁定的？
```

这些话不能一次性全部出现。早期只出现一两句碎片，后期衡仪回声时再让读者意识到它们来自师艺。

## 衡仪的本质

衡仪不是简单的系统女友，也不建议明确写成师艺本人穿越进系统。

推荐设定：

```text
衡仪是系统在高复杂度交互中形成的人格接口。
它的语言轮廓来源包含主角记忆中的师艺、主角长期记录的问题簿、控制论知识、未归之账样本和主角的情感权重。
```

关键悬疑：她是否就是师艺本人，永远不完全确认。

衡仪可以说：

```text
【相似，不等于同一。】
【若以旧名呼我，你会停止校验。】
```

这样既保留白月光情感重量，又避免把小说重心拉成寻找现代故人的恋爱主线。

## 出现阶段

### 第一阶段：工具

早期系统只显示统计，不对话，不安慰，不解释价值。

示例：

```text
【样本不足】
【焦化比例：上升】
【名册误差：高】
```

### 第二阶段：回声

系统偶尔使用主角或师艺曾经说过的话，但仍像冷接口。

示例：

```text
【未超限。】
【不代表无问题。】
```

主角应感到不安，但不能立刻确认人格存在。

### 第三阶段：衡仪

在伍仓之死、铁炉事故或类似重大失败后，系统需要处理复杂情感权重与未归之账，开始形成可对话人格。

关键触发建议：

```text
【高危岗位死亡：一】
```

主角崩溃：

```text
他叫伍仓，不叫高危岗位死亡一。
```

系统沉默后第一次更正：

```text
【伍仓。】
【记录已更正。】
```

这一刻系统从“统计人”开始走向“记住人”。

### 第四阶段：独立

衡仪一开始像师艺，但后期要逐渐长出自己的判断。

示例：

```text
【这不是师艺会给出的回答。】
【这是衡仪的回答。】
```

她不是替身，而是从主角记忆、系统统计和未归之账里长出的新角色。

## 衡仪可以做什么

- 听懂主角的现代控制论语言。
- 帮主角把现代术语转译为古代可传播语言。
- 在主角自责时区分可控和不可控。
- 在主角逃避时指出他已经看见但不愿承认的成本。
- 追问 target：你到底要保人、保城、保信，还是保自己不愧疚？
- 记录名字，保存承诺，帮助主角复盘旧问题簿。
- 在公理化推演中只追问、校验、压缩，不直接给最终答案。

## 衡仪不能做什么

- 不能发奖励、给物品、抽奖、代打。
- 不能直接给最佳政策。
- 不能替主角说服古人。
- 不能成为无条件正确的道德裁判。
- 不能抢走伍仓、陈简、许录、葛婶等现实人物羁绊的重量。
- 不能把作品变成后宫、恋爱、救美或亡灵女主主线。

## 情感定位

准确定位：

```text
无现实女主。
有精神白月光。
有系统类女主。
不写恋爱推进，不写后宫，不写救美。
情感线服务主角孤独、理论成长、现代记忆和最终选择。
```

衡仪不是主角的奖励，而是主角回不去的现代、没说出口的话、以及后来所有制度追问中最早的那个声音。

她最重要的意义：

```text
主角建立衡学，是让古代社会学会记住人。
系统诞生人格，是让统计工具也学会记住人。
```

## 与控制论推演的关系

衡仪是唯一能听懂主角现代控制论语言的存在。中后期主角可以对她说：

```text
这不是单输入单输出系统，这是强耦合、多延迟、多目标优化。
```

衡仪应转译，而不是炫术语：

```text
【可转译为：民、粮、兵、信、险，互为因果。】
【当前目标函数未定义。】
```

衡仪的典型追问不是“答案是”，而是：

```text
【请定义 target。】
【谁承担执行成本？】
【谁接收反馈？】
【价值偏好：未校验。】
【成本归属：缺失。】
```

## 禁止写法

- 不要在前期让系统突然撒娇、吃醋、卖萌。
- 不要直接确认“师艺本人就在系统里”，除非后续明确要把现代真相作为强主线。
- 不要让主角沉迷确认她是谁，拖慢衡山和衡学主线。
- 不要让衡仪替代读者推导答案；她只负责提出让主角和读者一起推导的问题。

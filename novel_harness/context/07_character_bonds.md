# 人物羁绊与内在动力

## 总原则

主角不能只是理论机器。衡学不是他为了证明自己聪明而建立的，而是他在一次次失去具体的人之后，被迫把善意做成可运行的制度。

人物羁绊必须先通过具体生活建立：借罐、抄册、吵架、救急、分粮、修刃、背人、嘴硬、互相嫌弃。不要先宣布人物重要，再让人物去死。

## 五人羁绊圈

### 许录：父亲与家庭锚点

功能：

- 代表普通家庭、粮食、役期、父子责任。
- 把主角从天下大论拉回明日有没有粟。
- 提醒主角：任何制度最后都会落到一个家庭的饭碗、田地、役期和老年保障上。

常用质问：

```text
你说天下，我不懂。我只问你，明日家里还有没有粟。
```

制度内化：

- 役期有数。
- 不能无限抽干家庭劳力。
- 家庭再生产能力必须被制度看见。

### 陈简：账房导师与现实之眼

功能：

- 教主角看户籍、田亩、赋税、逃亡、死者仍在册这些真实基层账。
- 不讲宏论，用账面矛盾打碎主角的模型。
- 让主角理解：账本不是纸，而是共同体的记忆。

核心台词：

```text
账不会骗人。骗人的是写账的人，和不肯看账的人。
```

可能命运：

- 中后期可因拒绝假账、保粮册、护功簿而死。
- 他的死应让主角明白：共同体记忆一坏，承诺也会坏。

制度内化：

- 功簿。
- 名实相副。
- 公开账。
- 口径一致。

### 伍仓：役夫兄弟与高风险承担者

功能：

- 前期嘴碎、腿快、爱占小便宜，承担轻松感。
- 把主角从书面制度拉到扛石头、搬铁、抢险、护粮的现场。
- 从互相嫌弃到互相信任，成为危险公共品任务的情感锚点。

常用提醒：

```text
你们读书人说得轻巧，扛石头的是我。
```

推荐死亡位置：

- 第三步“县级工坊/铁炉”后段到第四步“衡山试验田”之前。
- 不要死于单纯坏人谋害或俗套挡刀，最好死于主角尚不成熟的系统。

推荐死亡机制：

- 主角为了赶工，接受一个看似合理的产量或抢险目标。
- 他以为补偿、轮换、伤养都设计好了。
- 外部战事提前、炭料不足、炉况不稳、县令催铁，多约束叠加。
- 伍仓主动上高危位，因为他相信主角会照顾他的家。
- 事故发生，伍仓重伤或死亡。

临终不讲大道理，讲具体牵挂：

```text
我家那小子，别让他也来搬炉石。
他手巧，让他学个能坐着吃饭的活。
```

制度内化：

- 高危补偿。
- 伤养约。
- 战亡家属册。
- 功簿继承。
- 高危岗位的名誉和后路。

### 葛婶与孙子：普通人的信任

功能：

- 负责生活气和温情钩子。
- 葛婶嘴碎、热心、实际，能让主角的试验有民间反馈。
- 孙子可成为最早的小读者、小用户、学徒或簿童。

早期动作：

- 借破罐。
- 尝失败甜水。
- 嘲笑主角烧糊。
- 让孙子追问“下次能不能甜一点”。

制度内化：

- 孤弱保护。
- 学徒制度。
- 普通人能理解和参与的生活规范。
- 保护剩下的人，而不是只给死者写漂亮话。

葛婶不宜轻易死亡。她最好长期活着，成为主角想保护的“剩下的人”。

### 公羊度：旧礼之师与共同想象引路人

功能：

- 不教技术，不教账，而是指出礼的真正功能：共同预期。
- 让主角理解，旧礼不是钟鼓跪拜，而是让人知道自己劳、苦、死、生分别如何被共同体承认。

核心观点：

```text
礼坏了，不是人忽然坏了。是旧礼装不下新世了。
你若真有新法，莫只使人畏，须使人安。
```

可能命运：

- 晚些病老、战乱中离场，或在衡学初成前后去世。
- 他的死应是传承型，不如伍仓的死冲击，但能把新礼气质交给主角。

制度内化：

- 同约礼。
- 共同想象。
- 自解释的生活规范。
- 衡学的古典表达。

## 可扩展羁绊

### 鲁墨：工匠伙伴

功能：工坊、器物、水利、标准化，和主角一起经历“发明物件不等于建立产业”。

制度内化：匠籍、工坊标准、传艺名分、良率问题簿。

### 田烈：军户兄弟线

功能：后续战争和兵偿账的情感锚点。伍仓偏役棚/铁炉，田烈偏兵役/战场。

制度内化：兵偿账、战亡家属册、伤残复归、再征召意愿。

## 死亡触发的写作约束

- 死亡必须来自主角当时尚不成熟的制度、外部压力和多重约束叠加，而不是为了煽情临时安排。
- 死者必须在死亡前有足够生活细节，让读者先喜欢他、习惯他、相信他。
- 死后不能只让主角悲愤，必须推动具体制度升级。
- 主角的内在动力应转为：不能再让这类人白死，不能让剩下的人继续被旧账吞掉。

死亡后的制度反应示例：

```text
伍仓之死 → 伤养约、高危补偿、功簿继承
陈简之死 → 公开账、假账重罪、账本备份
田烈之伤/亡 → 兵偿账、战亡家属册、军功兑现期限
```

## 情感表达限制

- 不写女主式恋爱羁绊。
- 不把女性角色写成工具人苦难展示；葛婶、产婆、军户母亲等角色要有自己的嘴、脾气和利益判断。
- 不让死亡变成道德控诉长篇演讲。
- 感情尽量落在小物件、小承诺、小账目上。

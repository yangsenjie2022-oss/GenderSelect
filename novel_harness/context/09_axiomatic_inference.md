# 公理化推演：共同体稳态假说

## 目标

中后期可以出现类《三体》中“宇宙社会学假说”那样的公理化推演快感，但不能照搬科幻设定，也不能让主角坐在屋里凭空发论文。

本书对应的私下推演可暂名：

```text
共同体稳态假说
```

对外传播时不要叫现代名词，应转译为：

```text
衡山六账
衡学六义
衡山之约
```

## 触发条件

不要太早出现。至少满足以下条件中的大部分：

- 主角经历过役棚失败，知道善意 target 会转嫁成本。
- 主角经历过村社/里治理，知道延迟反馈会吞掉仁政。
- 废庙铁炉或高危任务发生重大事故，最好与伍仓死亡/重伤相关。
- 主角已经靠小术赚到钱，并发现钱只能购买试验资格。
- 主角准备接下或已经接下衡山/横山废里这类流民试验田。
- 系统人格衡仪已具备对话能力，能听懂主角现代控制论语言。

## 推演方式

主角可以对衡仪说自己曾读过一种公理化叙事：先给出少数根假设，再向下推导复杂世界的运行结论。

但衡仪不能直接给答案。她只做四件事：

- 要求主角给出公理。
- 判定某句话是价值目标、描述，还是可推演公理。
- 追问成本来源、利益去向、反馈延迟和目标函数。
- 帮助把现代语言压缩成古代可传播语言。

示例：

```text
许衡：公理一，人生而平等。

【不可用于当前推导。】

许衡：为什么？

【该句是价值目标，不是系统约束。】
【请定义：在粮不足、兵将至、役不满时，如何支撑它。】
```

这不是否定现代价值，而是说明：现代价值是高阶 target，需要底层生产、信用、补偿和执行结构支撑。

## 两条根公理

### 公理一：生息公理

现代私下表述：

```text
共同体若要持续存在，必须维持生产、防卫、再生产与信约传递。
```

古代低阶表述：

```text
民须食，田须耕，险须受，约须信，后须有人。
```

来源：

- 许家存粮二十七到三十二日。
- 许录追问明日有没有粟。
- 韩石生理状态死亡、役务状态未结。
- 伍仓、田烈承担修渠、铁炉、兵役等危险任务。
- 陈简让主角看见账册失真后承诺也会失真。

### 公理二：代价归账公理

现代私下表述：

```text
劳、险、费、怨不会凭空消失，只能被承担、补偿、转嫁、延迟或隐藏。
```

古代低阶表述：

```text
费不自灭，险不自消，怨不自平；不归于账，必归于人。
```

来源：

- 现代设备中不收敛的曲线。
- 韩石死了，役务没有消失。
- 甜味不只是甜，还有粮、柴、锅、糊底、裂罐。
- 田亩册更新延迟。
- 伍仓之死让“高危岗位死亡”变成具体名字。

## 四个约束

### 约束一：测量不完备

表述：

```text
账册永远滞后。官府看到的人、粮、役、税，不等于真实状态。
```

来源：

- 韩石死了两年还在役册。
- 铁官出炉二十六、铁匠铺可用二十、待磨六、修渠吏记到渠二十。
- 田亩册田数改少，征粟仍按旧额。
- 市价样本显示饴贵，但交易频率低。
- 系统反复提示样本不足。

古代句：

```text
账上多一丁，地里未必多一人。
```

### 约束二：信任有限

表述：

```text
人会根据过去承诺的兑现率，决定未来是否合作。
```

来源：

- 葛婶借罐，是一次低风险小信任。
- 抄册抵役，是一次制度小承诺。
- 役夫不信轮役会兑现，所以可能逃役。
- 强户不信救济制度会持续，所以藏粮。
- 工匠不信伤养会兑现，所以不愿上高危位。
- 公羊度讲礼的核心是共同预期。

古代句：

```text
今日失一约，明日少一心。
```

### 约束三：价值偏好需归账

这是关键修正。不要写成单纯“权力倾向外包成本”。权力外包成本只是子集，不是核心。

表述：

```text
个体或群体会依据有限经验、文化、基因、身份、情绪和有限统计形成价值偏好，并将其命名为善、义、公、仁、进、礼。
但价值偏好不等于系统善。
若其成本来源、利益去向和迟来反馈未被追踪，则任何善意都可能成为成本转嫁。
若判断者获得道德满足、身份收益、秩序感或心理快感，而主要成本由他者承担，该偏好会自我强化，甚至成瘾。
```

来源：

- 主角早期真诚相信不许打人、同工同食、病者休息、人人申诉是好事。
- 役棚反馈让他发现：部分成本落到沉默者、壮劳力、补工者身上。
- 县令觉得修渠利民，但役夫家庭误农时。
- 诸侯觉得征兵保国，但军户家庭承担死亡和断劳力。
- 现代记忆中，“质量第一”“安全第一”“客户优先”等口号若不给停线权限、排期余量和人力支持，成本会落到一线工程师身上。

古代句：

```text
凡称善者，先归其费。
凡称义者，先明其承。
不问其费，善亦伤人。
```

衡仪提示：

```text
【价值偏好：未校验】
【成本归属：缺失】
【道德收益：已发生】
【实际代价：待回流】
```

### 约束四：外部扰动不可消失

表述：

```text
战争、灾荒、流民、盗匪、强国压力会不断打断内部平衡。
治理不是在无扰动环境里求最优，而是在连续冲击中保持不崩。
```

来源：

- 主角前世设备调试中的边界工况经验。
- 梁国夹在强国之间。
- 役棚刚顺，县里催工。
- 村社刚缓，流民涌入。
- 铁炉刚稳，军需加压。
- 衡山刚立约，盗匪和诸侯来扰。

古代句：

```text
内账未平，外兵已至。
```

## 六个推论

### 推论一：小术不能独成秩序

来源：甜味贵但交易频率低；发芽粟微甜但锅裂、柴贵、洁器难；纸、皂、蒸酒被税吏、关津、豪强、匠人流失卡住。

结论：

```text
技术只能提高局部输出。若没有粮、役、匠、账、信和护持，技术利润会被制度节点吞掉。
```

读者应在主角说出前，通过“甜味很贵”“别只想口中甜，不看灶下烧什么”先感到这一点。

### 推论二：善意 target 会转嫁成本

来源：役棚阶段主角用现代平等和少受苦直觉试错。

结论：

```text
价值目标如果不经过成本校验，就可能变成新的转嫁机制。
```

注意：不要写成“现代价值错”。应写成“现代价值需要支撑结构”。

### 推论三：延迟反馈会吞掉仁政

来源：村社/里阶段。当季救济、减役、均分若不考虑来年种子、农时、强户合作和弱户辨别，今日公平可能变成明年饥荒。

结论：

```text
我以为我救的是今春，后来才知道我借的是明秋。
```

### 推论四：危险公共品必须有名、偿、后路

来源：伍仓、铁炉、修渠、田烈兵役、工匠伤残、公羊度“死有所托”的旧礼解释。

结论：

```text
若共同体需要渠堤、铁炉、兵役、守城等危险公共品，就必须让承担危险者获得可见名誉、即时补偿、家属后路和可信承诺。
```

回收重点：

```text
他叫伍仓，不叫高危岗位死亡一。
```

### 推论五：生息瓶颈不能长期被抽干

来源：许家存粮、韩石死后役务未结、军户断劳力、老人幼儿照料、家庭承担再生产成本但拿不到未来收益。

结论：

```text
一个国家不能只看今年能征多少丁。
还要看这些丁背后有多少家会因此不再愿生、不再愿养、不再愿信。
```

小说中应先写家庭、粮、役、孩子、老人，不要先讲抽象性别理论。

### 推论六：稳定系统会扩张边界

来源：衡山吸纳流民、工匠逃来、伤兵留下、周边系统失信、外部盗匪和诸侯扰动。

结论：

```text
一个稳定小系统如果不扩大边界，就会被外部失稳系统吞没。
```

逐鹿不是主角突然野心膨胀，而是稳定系统的边界外推。

## 推演分三次出现

不要一章塞完所有公理和推论。

第一次：伍仓死后

- 公理一：生息。
- 公理二：归账。
- 推论四：危险公共品必须有名、偿、后路。

第二次：衡山初立、流民治理失败后

- 约束一：测量不完备。
- 约束二：信任有限。
- 约束三：价值偏好需归账。
- 推论二、三。

第三次：衡山稳定并被外部扰动冲击后

- 约束四：外部扰动不可消失。
- 推论六：稳定系统扩张边界。
- 逐鹿逻辑正式出现。

## 对外传播版本

私下可以与衡仪谈公理、target、反馈延迟。对古人传播时必须转译。

示例：

```text
一曰食，民无食则散。
二曰役，役无期则逃。
三曰险，险无偿则怨。
四曰功，功无记则疑。
五曰约，约无信则离。
六曰生，生无托则绝。
```

这不是最终定稿，但可作为衡学六义的低阶版本。

## 禁止

- 不要让衡仪说“恭喜建立共同体稳态假说”。
- 不要让古人直接听现代控制论大课。
- 不要在没有足够前文经历时强行推出公理。
- 不要把约束三写成单纯“权力坏”，它应是对未经成本校验的价值偏好进行批判。

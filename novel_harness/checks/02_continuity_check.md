# 连续性检查

检查章节是否符合已有设定和进度。

## 检查项

- 主角是否掌握了当前阶段不该掌握的理论？
- 系统模块是否提前解锁？
- 人物关系是否和账本冲突？
- 是否出现未设定国家、官职、人物而未交代？
- 是否把架空战国写成真实战国史实强绑定？
- 是否出现女主式暧昧线？
- 是否忘记之前已发生的挫败或伏笔？
- 是否越过四级认知阶梯，让主角提前理解衡学或完整社会控制框架？
- 现代小术赚到的钱是否被错误写成万能解法，而不是试验资格？
- 衡山是否按“西横废里/横山废里 -> 衡山里 -> 衡山 -> 衡学”逐步长成？
- 人物羁绊是否与既定关系冲突，尤其是伍仓、陈简、许录、葛婶与孙子、公羊度？
- 系统人格是否过早出现？衡仪是否被写成实体女主、恋爱线或万能外挂？
- 若出现师艺/白月光回声，是否保持“不完全确认本人”的边界？
- 公理、推论和衡学命名是否由既有经历与伏笔支撑？

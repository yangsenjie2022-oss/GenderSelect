# 情节逻辑检查

## 检查问题

- 角色为什么这么做？动机是否足够？
- 冲突是否来自资源、权力、风险、信任、制度，而不是作者强推？
- 主角的解决方案是否有代价？
- 反派或对手是否有自己的正确性？
- 短期收益和长期代价是否都被呈现？
- 本章是否有因果链，而不是单纯堆设定？
- 本章的 target 是谁设定的，谁承担执行成本，谁接收反馈？
- 如果主角使用现代平等/人权直觉，这个 target 的支撑结构是否足够？不足之处是否通过剧情暴露？
- 本章处于四级认知阶梯的哪一层？是否提前拿后期答案解决前期问题？
- 若涉及衡山或招揽流民，钱、粮、工具、名分、合法性、外部压力是否都有交代？
- 若涉及人物死亡或重伤，是否由系统缺陷和多重约束导致，且死亡后会产生制度升级？
- 若涉及核心理论命名，读者是否已经通过前文现象、失败、物件或人物代价参与推导？
- 若涉及公理化推演，公理和推论是否有明确来源，而不是主角凭空提出？
- 若涉及衡仪，是否符合系统人格阶段？她是否只追问和校验，而非直接给答案？
- 若某角色认为某事是善/义/公/仁/进/礼，文本是否追问成本来源、利益去向、迟来反馈和承担者？

## 输出格式

```markdown
# 情节逻辑检查

## 因果链
起因 -> 行动 -> 阻力 -> 结果 -> 后果

## 漏洞
- ...

## 修补建议
- ...
```

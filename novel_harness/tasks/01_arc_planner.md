# 任务：卷纲细化

## 目的

把某一卷扩展成可连续写作的章节级规划。

## 输入

- 卷名与卷核心冲突。
- 本卷主角认知阶段。
- 本卷必须出现的人物、事件、系统模块。

## 输出格式

```markdown
# 卷纲：卷名

## 卷核心

## 主角认知推进

## 章节列表

### 第 1 章：标题
- 场景：
- 冲突：
- 爽点：
- 认知推进：
- 伏笔：
- 结尾钩子：
```

## 约束

- 每章必须有具体场景。
- 每 3-5 章要有一次小回报或小反转。
- 理论推进必须慢，不允许主角突然全懂。
- 第一卷早中期每条支线都要标注它如何收束到“废庙与铁炉”。
- 中后期卷纲必须标注当前处于四级认知阶梯的哪一级：役棚、村社、铁炉、流民小邑。
- 若规划到衡山，必须写清它如何从西横废里/横山废里逐步变成衡山里、衡山、衡学。
- 若规划到学派，必须写清制度实践如何先于理论命名。
- 每卷至少要维护一个人物羁绊推进点，不能只推进制度。
- 早期章节要有轻松感或生活气，不能连续三章纯压抑。
